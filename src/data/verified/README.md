# Verified providers (manual curation)

This folder is the ingestion point for **high-confidence** providers that should appear on indexed `/location/*` pages.

Why this exists:
- The automated standardized dataset can contain low-confidence/placeholder artifacts.
- Verified providers let us increase “realness” on top city pages safely (better trust + better rankings).

## How to add providers

1. Edit `src/data/verified/providers.json`
   - Manual curation goes here.
   - ISA credentialed providers live in `src/data/verified/providers.isa.json` (generated by scripts).
   - Auto-seeded “listed” providers live in `src/data/verified/providers.seeded.json` (generated by scripts).
   - Web-discovered “listed” providers live in `src/data/verified/providers.web.json` (generated by scripts).
2. Validate:
   - `node scripts/validate-verified-providers.mjs`
3. Publish:
   - `node scripts/publish.mjs --public-dir dist`

## providers.json format

`providers.json`, `providers.isa.json`, `providers.seeded.json`, and `providers.web.json` must be JSON arrays of objects. Minimum required fields:

- `locationSlug` (string) — must match a folder under `public_site/location/<slug>/`
- `slug` (string) — unique within the directory (used for stable internal IDs/links)
- `name` (string)
- `verification.sourceUrl` (string) — public URL that supports the listing
- `verification.verifiedAt` (string) — `YYYY-MM-DD` (for seeded entries this is the date we captured the public listing)
- One contact method: `website` OR `email` OR `phone`

Optional:
- `trust` (string) — `verified` or `listed` (defaults based on file)

Recommended fields (strongly suggested):

- `website` (string) — `https://...`
- `email` (string)
- `phone` (string)
- `address` (object): `city`, `region`, `country` (2-letter `US`/`CA` preferred)
- `specialties` (string[])
- `services` (string[])

## Example entry

```json
{
  "locationSlug": "kelowna",
  "slug": "example-valuation-inc",
  "name": "Example Valuation Inc.",
  "website": "https://examplevaluation.com",
  "email": "info@examplevaluation.com",
  "phone": "+1 250 555 1234",
  "address": { "city": "Kelowna", "region": "BC", "country": "CA" },
  "specialties": ["estate jewelry", "coins", "furniture"],
  "services": ["insurance appraisals", "estate settlement", "donations"],
  "verification": {
    "sourceUrl": "https://examplevaluation.com/services/appraisals",
    "verifiedAt": "2026-01-04",
    "notes": "Owner-confirmed via website contact page."
  }
}
```
